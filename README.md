Sistema de gestión de turnos
Aplicacion desarrollada en Python que permite gestionar turnos medicos mediante un sistema interactivo en consola.

Funcionalidades:
a)Asignacion de turnos
b)Consulta de turnos asignados
c)Organizacion de pacientes
d)Interfaz por menu interactivo

Logica implementada:
a)Menu interactivo con bucles
b)Validacion de datos
c)Estructuras condicionales
e)Uso de variables para almacenamiento de datos

Tecnologias:
Python - Visual Studio Code

Objetivo:
Aplicar logica de programacion en la simulacion de un sistema de turnos.
